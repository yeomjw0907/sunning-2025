**참고**
- 무역업무 서비스·재고관리의 업무절차 단계 이미지(`trade-step-*`, `inventory-step-*`)는 현재 화면에 표시하지 않습니다. 데이터/폴백용으로만 정의되어 있습니다.

## 공통 안내

| 항목 | 내용 |
|------|------|
| **폴더** | `public/images/documents/` |
| **확장자** | `.jpg` (기본). `.webp` 사용 시 `data/documentImages.ts` 에서 `const EXT = '.webp'` 로 변경 |
| **로컬 사용** | 이미지 넣은 뒤 `data/documentImages.ts` 상단에서 `USE_LOCAL_IMAGES = true` 로 설정 |

- **권장 가로**: 위 표 기준으로 준비하면 됩니다. 더 크게 넣어도 화면에서 축소되어 표시됩니다.
- **권장 비율**: 16:9(히어로), 4:3(일반)·1:1(원산지) 권장이며, 비율이 달라도 표시는 됩니다.
