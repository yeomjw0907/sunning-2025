# 서브페이지 이미지 가이드

서브페이지(무역업무, 시장조사, OEM/ODM, 원산지 표기 요금, 재고관리)에 사용할 이미지를 직접 넣는 방법입니다.

---

## 1. 폴더

이미지 파일을 아래 폴더에 넣습니다.

```
public/images/documents/
```

- Vite 빌드 시 `public` 폴더 내용이 그대로 배포됩니다.
- 위 경로에 넣은 파일은 사이트에서 `/images/documents/파일이름` 으로 접근됩니다.

## 3. 확장자

- **권장**: `.jpg`
- 사용 가능: `.webp`

기본 설정은 `.jpg` 입니다. `.webp` 만 사용하려면 `data/documentImages.ts` 에서 다음 한 줄을 수정합니다.

```ts
const EXT = '.webp';
```

---

## 4. 권장 크기

가로 기준으로 아래 정도면 충분합니다. 비율은 4:3 또는 16:9 를 권장합니다.

| 용도 | 권장 너비 (가로) |
|------|------------------|
| 히어로 이미지 (`*-hero.jpg`) | **1200px** |
| 무역/시장/OEM 섹션·단계 이미지 | **600px ~ 800px** |
| 재고 단계·원산지 6종 | **400px ~ 500px** |

더 큰 해상도로 넣어도 되고, 화면에서는 잘려 보이거나 축소되어 표시됩니다.

---

## 5. 로컬 이미지 사용하기

1. 위 **파일 이름**과 **폴더**에 맞춰 이미지를 준비한 뒤 `public/images/documents/` 에 저장합니다.
2. `data/documentImages.ts` 를 열고 상단 값을 바꿉니다.

```ts
const USE_LOCAL_IMAGES = true;
```

- **`true`**: `public/images/documents/` 에 넣은 이미지 사용
- **`false`**: 기존 Unsplash 이미지 사용 (기본값)

저장 후 다시 빌드하거나 개발 서버를 새로고침하면 반영됩니다.
